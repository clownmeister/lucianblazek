## Requirements

* Docker
* Make

## Setup

1) Run `make env`
2) Setup `.env` file and setup hosts accordingly
3) Run `make` or `make init`

## Assets

* Build assets with `make build`
* Start watching assets with `make watch`

## Other commands

For other commands check `./Makefile`
